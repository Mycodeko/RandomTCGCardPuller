Random TCG Card Puller

Random TCG Card Puller allows for drawing a random card from several trading card games such as Magic: The Gathering, Pokémon, and Yu-Gi-Oh!, as well as custom card sets in the Cockatrice XML format. Custom sets for several card games can be downloaded here.

FAQ

Q: Why do cards appear so slowly?

A: Due to relying on 3rd-party APIs with varying max requests limits, a safe value of 1 API call per second was chosen to ensure a temporary IP block does not occur.

Q: How do I import a custom card set?

A: In the game select bar, choose --Import Game--, then select a valid Cockatrice card set XML file. The game will appear on the list if the import was successful.

Q: Why are the cards sometimes purple rectangles?

A: If the image could not load for the card, a purple rectangle will be shown instead.
